# OpenCL matrix multiplying (naive realisation)


# Build
Windows:
```
>mkdir build
>cd build
>cmake ..
>cmake --build ..
>MatrixMult.exe
```
Linux:
```
>mkdir build
>cd build
>cmake ../
>cmake --build ../
>./MatrixMult
```

# Test
Windows:
```
>MatrixMult.exe -t 
```
Linux:
```
>./MatrixMult -t
```
